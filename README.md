# 设计师作品集网站

一个现代、极简的作品集网站，用于展示设计作品，使用HTML5、CSS3、JavaScript和GSAP动画构建。

## 功能

- 简洁、极简的设计，专注于作品本身
- 响应式布局，适用于所有设备
- 使用GSAP实现流畅的动画和过渡效果
- 可筛选的作品画廊
- 详细的项目页面
- 关于部分，包含经验和教育的时间线
- 联系表单
- 移动友好的导航

## 使用的技术

- HTML5
- 使用Tailwind CSS的CSS3
- JavaScript
- GSAP (GreenSock动画平台)
- Font Awesome图标

## 项目结构

```
/
├── index.html          # 主作品集页面
├── works/              # 项目详情页面
│   ├── work1.html
│   └── ...
├── css/                # CSS文件
│   └── style.css
├── js/                 # JavaScript文件
│   └── script.js
└── assets/             # 图片和其他资源
    └── images/
```

## 使用方法

1. 克隆或下载仓库
2. 在网络浏览器中打开 `index.html`
3. 要查看项目详情，请点击画廊中的任何作品

## 定制

- 将 `assets/images/` 中的图片替换为您自己的作品
- 更新 `index.html` 和作品详情页面中的文本内容
- 在Tailwind CSS配置中修改配色方案
- 在JavaScript文件中调整动画
