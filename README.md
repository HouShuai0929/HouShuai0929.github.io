# 侯帅作品集网站

一个现代、极简的个人作品集网站，用于展示个人项目和技能，使用HTML5、Tailwind CSS、JavaScript和GSAP动画构建。

## 功能特性

- 简洁、现代的设计风格
- 响应式布局，适配所有设备尺寸
- 流畅的滚动和过渡动画
- 作品展示画廊
- 个人信息和技能展示
- 联系表单
- 移动端友好的导航菜单
- 回到顶部按钮

## 技术栈

- **前端框架**：HTML5 + Tailwind CSS
- **JavaScript**：原生JavaScript
- **动画库**：GSAP (GreenSock Animation Platform)
- **图标库**：Font Awesome

## 项目结构

```
/
├── index.html          # 主页面
├── css/                # CSS文件
│   └── index.css       # 外部CSS样式
├── js/                 # JavaScript文件
│   └── index.js        # 外部JavaScript逻辑
├── img/                # 图片资源
│   ├── houshuai.jpg    # 个人照片
│   └── jingdong.jpg    # 项目图片
└── README.md           # 项目说明文档
```

## 快速开始

1. **克隆或下载**本仓库到本地
2. **直接打开** `index.html` 文件在浏览器中查看
3. **浏览**网站内容，点击作品卡片查看详情

## 自定义指南

### 更新内容
- 修改 `index.html` 中的个人信息、技能和项目描述
- 替换 `img/` 文件夹中的图片为您自己的照片和项目图片

### 调整样式
- 修改 `css/index.css` 中的自定义样式
- 在 `index.html` 的 Tailwind 配置中调整配色方案

### 修改功能
- 在 `js/index.js` 中调整动画效果和交互逻辑
- 添加或移除页面 sections 以适应您的需求

## 响应式设计

网站采用响应式设计，在以下设备上均有良好表现：
- 桌面端（1200px+）
- 平板端（768px-1199px）
- 移动端（360px-767px）
